Attunement(https://attunement.io)

Attunement is a card database and advanced search engine I built for the Legends of Runeterra collectible card game. It’s written in Haskell using Servant and Warp. It’s backed by a PostgreSQL database and uses DigitalOcean’s S3-like Spaces for image storage. The front-end is almost entirely server-rendered HTML, with some SPA-like features in the admin console using HTMX.
